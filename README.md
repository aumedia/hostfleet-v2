# HostFleet v2 — Ship AI Apps. Everywhere.

Independent benchmarks of AI deployment infrastructure: GPU inference hosts, edge runtimes, and AI-generated site hosting.

- Stack: Astro + React + Tailwind v4
- Deploy: Cloudflare Pages
- Editor persona: Alex Harmon, Senior Infrastructure Analyst

## Local
\`\`\`
npm install
npm run dev
npm run build
\`\`\`

## Adding posts
Drop \`.md\` files in \`src/content/posts/\` with the frontmatter schema in \`src/content/config.ts\`.
